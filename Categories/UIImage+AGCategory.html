imageWithContentsOfBundleFileNamed_AG:

Creates and returns an image object by loading the image data associated with the specified filename in the main bundle.

+ (UIImage *)imageWithContentsOfBundleFileNamed_AG:(NSString *)filename

Parameters

filename

The name of the file, including extension.

Return Value

A new image object for the specified file, or nil if the method could not initialize the image from the specified file.

Discussion

This method does not cache the image object.

Parameters

Return Value
