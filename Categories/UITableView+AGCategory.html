scrollToRow_AG:inSection:

Scrolls the receiver to the specified row in the specified section.

- (BOOL)scrollToRow_AG:(NSInteger)row inSection:(NSInteger)section

Parameters

row

An index number identifying a row in a section of a table view.

section

An index number identifying a section of the table view.

Return Value

YES if the receiving table view will scroll, otherwise NO.

Discussion

The receiver will not be scrolled if the index path constructed from the specified row and section is found to be invalid.

scrollToRow_AG:inSection:atScrollPosition:animated:

Scrolls the receiver until the specified row in the specified section is at a particular location on the screen.

- (BOOL)scrollToRow_AG:(NSInteger)row inSection:(NSInteger)section atScrollPosition:(UITableViewScrollPosition)scrollPosition animated:(BOOL)animated

Parameters

row

An index number identifying a row in a section of a table view.

section

An index number identifying a section of the table view.

scrollPosition

A constant that identifies a relative position in the receiving table view (top, middle, bottom) for row when scrolling concludes.

animated

YES if you want to animate the change in position, NO if it should be immediate.

Return Value

YES if the receiving table view will scroll, otherwise NO.

Discussion

The receiver will not be scrolled if the index path constructed from the specified row and section is found to be invalid.
